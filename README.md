## Overview

Advanced survival analysis of neonatal mortality in a synthetic cluster-randomised controlled trial (N = 5,000) modelled after JiVitA-style maternal and child health surveillance in rural Bangladesh. This project demonstrates publication-grade statistical methods and visualisations for a prospective cohort followed from ~20 weeks gestation to 12 months postpartum.

---

## Study Design

| Parameter | Detail |
|-----------|--------|
| Study type | Cluster-randomised prospective cohort |
| N participants | 5,000 pregnant women (Simulated) |
| N clusters | 50 villages |
| Districts | Gaibandha, Nilphamari, Lalmonirhat, Kurigram, Rangpur, Sirajganj |
| Follow-up | Enrolment (~20 wk gestation) → 12 months postpartum |
| Primary outcome | Neonatal mortality (death within 28 days of birth) |
| Secondary outcomes | Low birthweight, preterm birth, maternal haemoglobin trajectories |
| Interventions | Control · BEP Supplement · MNP · BEP + MNP |

## Methods

### Survival Analysis
| Method | Purpose | R Package |
|--------|---------|-----------|
| Kaplan-Meier curves | Non-parametric survival estimation | `survival`, `survminer` |
| Log-rank test (Peto-Peto weighted) | Group comparison | `survival` |
| Cox PH regression | Adjusted hazard ratios | `survival` |
| Schoenfeld residuals | PH assumption testing | `survival::cox.zph` |
| Gamma frailty model | Cluster random effects | `survival` |
| Fine-Gray subdistribution HR | Competing risks | `cmprsk` |
| Restricted Mean Survival Time | Treatment effect (RMST) | `survRM2` |
| Kernel-smoothed hazard function | Hazard visualisation | `muhaz` |

### Reporting
- `gtsummary` — publication-quality regression and baseline tables
- `ggplot2` + `patchwork` — multi-panel composite figures
- `plotly` — interactive visualisations in Shiny

## How to Run

```r
# 1. Install required packages
install.packages(c(
  "tidyverse","lubridate","survival","survminer","broom",
  "gtsummary","gt","ggpubr","patchwork","scales","RColorBrewer",
  "ggtext","cmprsk","muhaz","survRM2","simsurv","kableExtra",
  "shiny","bslib","plotly","DT","fs"
))

# 2. Set working directory to project root
setwd("path/to/survival_analysis")

# 3. Generate dummy data locally
source("R/01_generate_data.R")

# 4. Run analysis & generate all figures
source("R/02_survival_analysis.R")

# 5. Launch interactive Shiny dashboard
setwd("shiny_app")
shiny::runApp("app.R")
```

## Key Results (Synthetic Data)

| Outcome | Rate |
|---------|------|
| Neonatal mortality | ~42 per 1,000 live births |
| Low birthweight | ~21% |
| Preterm birth | ~16% |

### Adjusted Cox PH — Neonatal Mortality (select covariates)

| Covariate | Adjusted HR | 95% CI | p-value |
|-----------|-------------|--------|---------|
| BEP Supplement vs Control | ~0.61 | 0.48–0.77 | <0.001 |
| BEP + MNP vs Control | ~0.52 | 0.41–0.67 | <0.001 |
| Low birthweight | ~3.2 | 2.6–4.0 | <0.001 |
| Preterm birth | ~2.5 | 1.9–3.2 | <0.001 |
| Severe anaemia vs None | ~1.7 | 1.2–2.5 | 0.003 |
| Skilled birth attendant | ~0.52 | 0.40–0.68 | <0.001 |

⚠️ **Important Note:** All values and hazard ratios listed above are approximate and simulated using the `simsurv` package for portfolio demonstration. They do not represent actual clinical trial findings.
